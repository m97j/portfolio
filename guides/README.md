### 📌 표기 규칙
- **형식**: `프로젝트명<대분류[세부분류]>`
- **대분류 Tag**: 기술 영역을 나타내는 도형·색상 이모지
- **세부 Tag**: 해당 프로젝트의 핵심 기술/연구 분야를 나타내는 의미형 이모지
- **멀티태그**: 세부분류를 2개 이상 붙일 수 있음 (예: `<🔴[🖇🎨]>` → Multimodal AI + Generative AI)
